# Production Landing Page

A vintage-styled landing page for the "30-Day Single" music production service.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Features

- Vintage/analog aesthetic with torn paper edges
- Interactive audio player component (Seed/Harvest toggle)
- Smooth scroll animations
- Responsive design
- Film strip timeline visualization
